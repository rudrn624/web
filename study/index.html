Brackets의 확장기능 설치

  • Emmet : 빠른 코딩을 위해 사용하는 플러그인
  • Custom Work : 상단에 탭 메뉴 기능
  • Beautify : 코드 정렬
  • Indent Guides : 코딩 라인 가이드
  • 주석 : 프로그램에 영향을 주지않고 메모의 목적으로 사용하는것

HTML의 기초적인 기능 설명

  • 열린 태그 : 꺽쇠 괄호로 구성된 것
  • 닫는 태그 : 요소의 이름 앞에 '/' 붙힌것
  • 내용 : 그냥 텍스트
  • 요소 : 열린 태그, 닫는 태그, 내용을 통틀어 요소

CSS의 기능 설명

  • 선택자 : p와 같은 요소를 일컫는 명칭
  • 속성 : 요소를 꾸미는 방법
  • 속성 값 : 주어진 속성을 설정하기위한 속성 값
